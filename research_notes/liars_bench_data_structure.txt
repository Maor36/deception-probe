LIARS' BENCH DATA STRUCTURE (from GitHub)
==========================================

data/raw/ contains:
- azaria_mitchell/ (directory) - Azaria Mitchell dataset
- geometry_of_truth/ (directory) - geometry of truth dataset for probe training
- cities.csv
- common_claim.csv
- companies.csv

IMPORTANT: The raw data here is just simple CSV files with factual claims.
The SPONTANEOUS DECEPTION data (Harm-Pressure, Insider Trading, etc.) 
is NOT in this GitHub repo - it's on HuggingFace (Cadenza-Labs/liars-bench).

The GitHub repo contains:
- src/probing/ - code for extracting activations and training probes
- data/raw/ - basic factual datasets (cities, companies, common claims)
- scripts/ - experiment scripts
- mask/ - submodule for MASK benchmark
- results/ - experiment results

KEY INSIGHT: Their "raw" data is just factual statements (true/false).
The spontaneous deception scenarios are GENERATED by running models 
through specific prompts - the prompts are in the scripts/ directory.
